## **Code Challenge:**
Write a programme that determines if any given binary number is divisible by 10

#### **Constraints:**
- None

----

### Code Examples

```python
# 1.0.0
x = int(input(), 2)

if not x%10:
    print(f"{x} can be divided by 10")
else: 
    print(f"{x} can not be divided by 10")
```
